# Tenis Oyunu 🎾

Bu proje, Pygame ile yapılmış basit bir 2 kişilik tenis oyunudur.

## Nasıl Oynanır?

- Sol oyuncu:
  - Yukarı: W
  - Aşağı: S

- Sağ oyuncu:
  - Yukarı: Yukarı ok
  - Aşağı: Aşağı ok

## Özellikler

- Skor sistemi
- Top fiziği
- Basit arayüz
- ayrıyeten top rastgele birinde yalpalanır eğer 1.oyuncuda yalpalanırsa 's' tuşu 2. oyuncuda yalpalanırsa 'down' tuşu kaybetmekten  kurtarır 

## Gereksinimler

- Python
- Pygame

python oyun.py
